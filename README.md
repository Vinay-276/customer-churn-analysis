Telecom Customer Churn Analysis

Problem Statement

Telecom companies face high customer churn, leading to revenue loss.

Objective

* Analyze churn behavior
* Identify high-risk customers
* Measure revenue loss

Tools Used

* SQL
* Power BI

Key KPIs

* Total Customers: 7043
* Churn Rate: 27%
* Revenue Leakage: ₹139K

Key Insights

* Month-to-month customers churn more
* Early customers are high risk
* Customers without Tech Support churn more

Impact

Reducing churn by 5% can save approximately ₹3L per year
